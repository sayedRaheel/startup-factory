# Vigil

Vigil (`agtop`) is a lightweight, brutally fast terminal UI for monitoring AI agents. The AI space is drowning in bloated web dashboards, so we are going back to the roots. It provides real-time telemetry using WebSockets and in-memory data structures.

### Problem Statement
Monitoring autonomous AI agents is currently overly complex, requiring full-stack observability platforms that persist every thought to a database. Developers need a zero-configuration, ephemeral, immediate-mode monitor that can sit side-by-side with their agent code.
